# 介绍

每当我进行一次测试时，总有几种方法接近我的目标。请注意我是如何提到圣灵测试和无虫奖励的，因为这是两个完全不同的东西。在我看来，Pentesting 可以更容易地找到漏洞，但不要因此而打消你寻找漏洞的念头，因为虽然 pentester 可能会找到更多问题，但其中许多问题的优先级较低或影响较小，而且好像这还不够，甚至 pentester 也需要非常完整。

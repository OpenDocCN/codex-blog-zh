# 主办虚拟工程峰会

*虚拟峰会是建立跨时区团队合作的有效方式*

自从疫情启动以来，许多企业已经从 T2 转移到远程办公，结果令人鼓舞。在可预见的未来，许多软件开发人员可能会生活在一个完全遥远的世界。我们如何保持团队之间的联系？

解决这个问题的传统方法是面对面的会议。您可以在[有吸引力的当地会议中心](https://www.bellharbor.com/)或酒店预订房间；或许邀请世界各地的人加入一个技术团队

在 Hired 的 2022 年软件工程师状况报告中，Python 排名第一，其次是 JavaScript、Java、TypeScript 和 C#。这种崇拜背后的原因是什么？《雇佣》杂志还就这个问题对软件工程师进行了调查:

*   生态系统
*   我很了解它
*   学习和发展资源易于获取
*   这是一种有趣且容易编程的语言
*   来自社区的支持
*   Python 是高需求

'mail':'vivek@email.com'}

这里，我们已经创建了一个具有姓名和电子邮件地址的用户，但是如果我们想要向这个字典添加更多的用户，简单的解决方案是，我们可以使用字符串列表而不是字符串作为值。

user = {'name':['vivek','alka'],
'mail':['vivek@email.com','alka@email.com']}

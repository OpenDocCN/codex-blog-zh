# 自我维持的软件团队

最好的团队是没有领袖也能运作的团队。

不是因为他们没有领袖，而是因为他们都是领袖。

他们不会参加所有的会议，也不会参与所有的决策，但是他们有自己的领导和负责的领域。

这在理论上听起来很棒，但在实践中，有大量的工作要做…

## 了解 Python 中字符串表示方法的区别

![](img/451941f2bf23227c94d09b6f73864918.png)

`str()`和`repr()`方法都用于对象的字符串表示。`str()`和`repr()`的区别在于`str()`返回一个对象的用户友好版本。`repr()`方法返回一个对象的开发人员友好的字符串表示。

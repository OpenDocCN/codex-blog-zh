在本教程中，我们将讨论 Tableau 中的不同等级函数，并演示如何使用它们。在此之前，让我们先讨论一下什么是等级。等级有助于给我们一个项目的有序划分或每个项目在层次结构中的位置。在 Tableau 中，我们可以使用 Rank 来计算数据集中任何度量(数字相关)的顺序。

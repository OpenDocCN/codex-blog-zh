本文探索了用于 Kubernetes 的集成开发环境 [Lens](https://k8slens.dev/) ，它可以简化集群的管理、监控和可观察性。它是一个独立的应用程序，可以在所有平台上使用。

Lens 是相当强大的，它为我们提供实时统计、日志流和开箱即用的故障排除功能，同时它是开源和免费的。

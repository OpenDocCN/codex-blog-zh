什么是附加变量活动

追加变量活动 是用来将值添加到类型数组中的现有变量只有中定义的数据工厂流水线 。

先决条件

阅读文章 关于 如何使用设置变量活动 创建管道。

第二步 - 点击Azure数据工厂资源ADF-Oindrila-2022-3月 。

另外 增加 一个 变量追加数组类型 。做 不提供 任何 默认值 。

元组是一种非常类似于列表的数据结构。它们与列表的区别在于它们的内容不能改变。因此，元组是不可变的。我们来看看元组的性质。

*   元组是有序的，可以通过索引访问
*   元组不能修改，之后不能添加或删除项目
*   元组在括号()中定义，所有元素用逗号分隔。
*   元组可以存储不同类型的数据。(字符串、整数、浮点、列表等。)
*   它们可以有两个值相同的项目，因此可能有重复的数据。

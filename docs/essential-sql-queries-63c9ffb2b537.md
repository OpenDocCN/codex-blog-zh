# 基本 SQL 查询

> 原文：<https://medium.com/codex/essential-sql-queries-63c9ffb2b537?source=collection_archive---------18----------------------->

您可以使用任何有效的名称在数据库中创建表格，包括数字和字母。该名称应该至少有四个字符长，并且应该以字母开头。如果表名太长或容易混淆，还应该在表名前面加上别名。如果不显式引用该表，查询将会失败。您还可以创建自定表格名称，如“mytable”，以简化创建自定表格的过程。

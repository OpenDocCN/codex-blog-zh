# 哪种编程语言？

![](img/3b5fb505c7e24d3dcda0ee1777e17aee.png)

基金会——照片由[布雷特·乔丹](https://unsplash.com/@brett_jordan?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

正如维基百科所说，大约有 700 种编程语言！有这么多选择，从中挑选一个是很有挑战性的。

我建议你从 Python 开始——因为它可读性强，非常适合学习编程的一般流程。这是我的第一语言，也将永远是我的最爱。它的易用性、便携性等…是无与伦比的。

另一个好处是大公司绝对喜欢它！例如，谷歌在 Youtube 等许多网站上使用 Python，甚至将其命名为谷歌的官方语言。

下载:[https://www.python.org](https://www.python.org)

## 数据科学

尽管我的上一篇文章[(不买电动汽车的 5 个理由)](/codex/5-reasons-not-to-buy-an-electric-car-2da605ade925)，数据科学(IMO)最令人印象深刻的用途是电动汽车确保其电池有效使用和可靠运行的能力。这一切都是通过结合预测分析和数据智能的力量实现的！

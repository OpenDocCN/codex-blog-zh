# DX(开发者体验):什么是？

![](img/d3b7490cfaeff578c97fc545266f40e2.png)

DX 是软件开发行业中的一个术语，用来描述软件使用者的体验。该术语最初是由微软在 21 世纪初创造的，并已在行业中广泛使用。DX 很重要，因为它会影响用户对软件的感觉，并且会影响他们是否继续使用它的决定。

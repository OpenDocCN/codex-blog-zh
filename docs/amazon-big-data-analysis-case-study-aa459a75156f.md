## 亚马逊大数据分析案例研究

大数据是一种高级技术，主要用于评估和整合公司中收集的数据。大数据的使用正在增加，许多公司正在使用大数据的关键功能来提高业务和开发系统的性能。亚马逊是向消费者提供各种服务和产品的领先电子商务组织之一[1]。主要目的是研究大数据在亚马逊的重要性，并评估大数据在亚马逊电子商务行业背景下的关键方面。本文主要关注三个重要部分:大数据的当代元素、数据特征和大数据准备。

# 安全问题和挑战

亚马逊是最大的电子商务行业，它以一种有效的技术向客户和大数据提供许多服务，亚马逊使用这种技术来提高数据收集流程的性能和有效性。从当代章节中选择了两个重要方面，包括安全问题和挑战[5]。人们发现大数据可以减少业务的复杂性和错误，亚马逊正在使用这种技术来提高他们的绩效。安全性是与亚马逊使用的大数据方法相关的重要问题之一。犯罪分子向中央服务器发送不想要的信号，并收集所有可靠的数据[1]。最近的一项研究发现，由于服务器和计算网络中存在未经授权的访问，数据安全对亚马逊来说是一个挑战。

然而，三个主要风险因素与大数据相关联，增加了亚马逊行业的安全相关问题，如缺乏安全性、利用未经授权的网络以及服务器的错误配置。因此，声明亚马逊在他们的业务中使用大数据技术的同时，应该关注安全性。根据 Bello、Jung 和 Camacho (2016)的研究，大数据技术中存在各种安全问题和威胁，如 DDOS 攻击、恶意软件活动、虚假数据生成、数据泄露问题和钓鱼攻击等[1]。DDOS 和恶意软件是与大数据相关的无处不在的安全威胁，它们直接影响了亚马逊的安全，降低了亚马逊使用的计算网络的性能。

前期调查发现，大部分犯罪分子利用不真实的网络和恶意工具，产生大量的流量信号和网络。这些信号从一个网络传输到另一个网络，并影响数据的隐私[6]。据了解，大数据使用互联网连接和计算网络来执行数据集成活动。尽管如此，有时用户使用第三方应用程序从消费者那里获得可靠的数据，这直接影响隐私并导致网络安全攻击。此外，有人认为，利用大数据技术，控制和管理数据隐私是亚马逊面临的另一个挑战。根据 Bowker (2014)的说法，网络配置错误是增加亚马逊数据泄露几率的一个关键问题。开发人员无法识别在工作场所产生挑战和风险的关键因素[2]。

控制和管理来自亚马逊使用的 IT 系统的恶意软件活动和信号是另一个挑战，这有助于犯罪分子在未经许可的情况下进入亚马逊的主服务器并收集数据。此外，亚马逊行业面临着与大数据相关的其他几个挑战，例如，未经授权的网络访问、数据集成、数据保护问题、数据的保密性、监控系统中的虚假数据等等。因此，有人认为，在使用大数据的关键功能时，亚马逊应该关注与隐私相关的问题，并制定适当的安全计划来保护收集的数据。此外，通过向员工提供适当的培训和教育，并在使用大数据分析方法的同时评估有效的数据集成技术，可以解决与数据集成相关的问题[9]。

# 数据源和数据格式

从数据方面，选取了两大因素，包括亚马逊电子商务行业背景下的数据源和数据格式。有人认为，大数据是一种有效的计算技术，可以适当地集成结构化和非结构化数据。最近的一项研究表明，亚马逊现在正在向消费者提供基于云的服务，通过这些服务，小型企业社区可以提高计算系统和网络的性能。据了解，数据格式是亚马逊用于评估从各种资源收集的数据的大数据技术的重要组成部分[10]。

而且，亚马逊在他们的业务中使用各种文件格式，比如文本文件、序列文件、Avro 数据文件等。在大数据技术的帮助下，数据可以被有效地格式化，亚马逊可以降低系统的复杂性，提高网络的性能。这种数据格式有助于开发人员从众多来源收集数据或事实，并使用大数据分析方法有效地评估每个数据。根据陈和张(2014)的研究，大数据可以提高数据集成流程的性能，并通过有效地监控数据格式来帮助亚马逊提高生产率[3]。

在使用大数据分析技术的同时，亚马逊可以有效地评估半结构化和非结构化数据，并降低系统的复杂性。在 Hadoop 中存储数据时，Amazon 需要识别假数据生成和不需要的信号，这些信号可能会直接影响计算设备和服务器的性能。众所周知，亚马逊和大数据技术涉及三种主要的数据文件格式，例如，优化的行分栏、拼花和 Avro [11]。所有这些数据格式都可以存储亚马逊的数据文件，帮助管理层减少与数据格式相关的问题。在大数据过程中，以这种数据格式记录的文件可能会被拆分到不同的磁盘上，并帮助亚马逊行业提高数据的可扩展性和可用性。

所有这些数据格式主要承载计算机文件中的数据，并提供一个平台，亚马逊可以轻松有效地评估收集的数据。数据格式的主要问题是，管理层需要有效的方法将可靠的数据存储到数据中心，从而增加了系统的复杂性[12]。amazon 主要使用面向列的数据格式来存储和记录消费者和员工的数据。尽管如此，建议管理层在使用大数据技术的同时包括基于行的存储系统，因为它们能够减少系统错误。

有人认为，数据源是大数据的重要组成部分，它提供了一个平台，亚马逊可以在这个平台上收集和评估与客户和利益相关者相关的数据。Chong，Chong，Liu and，Li (2017)指出，亚马逊主要使用第三方应用程序和社交媒体网站来获取与消费者相关的数据，并使用大数据网络以文件格式存储这些数据[4]。而且 AWS 是亚马逊提供的有效服务之一，帮助其他小企业社区。据观察，大数据技术帮助亚马逊在很短的时间内收集数据。管理团队使用各种数据源，包括脸书、Twitter、在线社区、数据驱动系统和其他 IT 资源。

社交互动是亚马逊和其他电子商务公司常用的来源，因为它在很短的时间内提供了许多数据集，包括结构化和非结构化数据[13]。在当今时代，大多数消费者在互联网上搜索时都在使用社交网络和应用程序。亚马逊使用社交网络作为数据源来收集海量数据，然后使用大数据技术进行整合。结论几个电子文件可以作为亚马逊的数据来源，有助于获得可靠的数据，包括与消费者和利益相关者相关的信息。因此，声明亚马逊采用的数据源有助于获得有效和可靠的数据，但有时黑客会产生虚假数据，这可能会在系统中产生冲突，并在大量的数据分析过程中出错。

# 去规范化和数据集成

大数据准备有许多方面，例如，去规范化、数据集成、聚合和数据清理。在这篇研究论文中，将在亚马逊电子商务行业的背景下解释去规范化和数据集成这两个方面。大数据中的术语标准化是亚马逊用于标准化和评估收集的数据以提高业务整体绩效的一种流程。Erevelles、Fukawa 和 Swayne (2016)认为，去规范化被定义为提高开发的数据库的读取性能并减少系统的错误[5]。据观察，亚马逊是一个领先的电子商务行业，为消费者提供 AWS 服务。管理层需要一个去规范化流程以及大数据来评估之前获得的数据集。

这种数据准备可以增强 Amazon 使用的数据仓库的性能，并为组织提供一个可以存储大量数据集的平台。与去规范化相关的主要问题是，如果开发的系统在数据库系统中不能正常工作，它可能会影响 Amazon 的性能[14]。此外，当在业务中开发和实现这样的过程时，Amazon 可以在数据模型和查询模型之间产生有效的关系。在去规范化过程的帮助下，亚马逊可以轻松减少大数据系统中与完整性相关的问题。尽管如此，它仍然需要服务器和数据集之间的适当通信。

在过去的几年中，Amazon 的管理团队改变了他们的数据库系统，并加入了去规范化的关键方面，以提高所提议系统的有效性。在审查了最近的报告后，发现亚马逊在当前系统中采用了大数据技术和去标准化流程，为减少开发人员面临的问题和复杂性提供了更好的平台[15]。网络和数据集之间缺乏适当的资源和通信可能会加快退化速度，影响已开发系统的性能。

因此，有人指出，在使用大数据去标准化的同时，亚马逊应确保他们关注网络和退化问题，并确定可能产生违规和数据丢失相关问题的风险因素。哈什姆等人(2015)提供了他们对大数据的看法，并建议在大数据的背景下，数据建模的更好选择是去规范化，这不需要更多的时间，并减少了系统浪费时间的比率[6]。因为这样的好处，亚马逊使用了去规范化和大数据技术的关键方面。

根据 Kim、Trimi 和 Chung(2014 年)的说法，术语数据集成是大数据的一部分，它结合了用于有效评估和集成存储数据的业务和技术流程[7]。亚马逊公司主要使用数据集成方法和大数据来评估非结构化和半结构化数据。这样一个过程为 Amazon 提供企业相关的服务，并帮助在各种数据集中扩展所获得的数据，通过这些数据，管理层可以轻松有效地存储数据。

最近的文献表明，数据集成为 Amazon 提供了广泛的数据质量能力。管理层可以监控和改进从各种来源收集的数据的质量。数据整合是亚马逊和其他商业公司可以整合和评估大量数据集并监控收集数据有效性的最佳平台之一。在大数据的帮助下，技术亚马逊现在正在进行数据整合相关的活动。尽管如此，虚假数据生成是犯罪分子制造的一个关键问题，无法轻易解决，亚马逊可能会受到安全相关问题的困扰。据分析，大数据集成在电子商务行业中发挥着主要作用，在亚马逊也用作数据保护工具。一般来说，这种过程结合了数据来源和软件格式。然后，它向消费者提供累积数据的未定义视图，这可能会在亚马逊行业产生问题。

对亚马逊来说，控制和管理数据整合是一个复杂的步骤，因为它需要更好的决策方法和可靠的系统来评估收集的数据。根据 Kiran，Murphy，Monga，Dugan 和 Baveja (2015)的说法，亚马逊在使用数据集成流程和大数据时面临各种挑战。这些挑战包括数据不确定性、发现洞见、数据可用性以及跨数据资源同步[8]。数据的不确定性是直接影响亚马逊使用的数据库和系统的一个主要问题。大多数犯罪分子向亚马逊发送不真实的网络和虚假数据，以进入数据集并收集所有消费者的敏感数据。

因此，声明去规范化和数据整合是亚马逊使用的有效数据准备方面，有助于评估来自数据源的可靠和有效的数据集。

# 结论

在回顾了亚马逊环境中的大数据方面后，得出的结论是，大数据是一种有效的技术，它提高了亚马逊的性能，并在很短的时间内监控了大量数据。这项研究批判性地回顾了大数据方面和网络在亚马逊电子商务行业中的作用。它强调了与大数据和亚马逊网络相关的安全问题。发现 DDOS 和恶意软件是大数据技术中普遍存在的安全威胁，影响了亚马逊的个人数据及其性能。虽然使用大数据的关键方面，亚马逊可以提高其性能，并有效地评估非结构化和半结构化数据。不过，它需要先进的计算网络和大空间来存储消费者的数据。

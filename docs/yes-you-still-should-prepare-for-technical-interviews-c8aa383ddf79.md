# 是的，你仍然应该准备技术面试

来源: [Unsplash](https://unsplash.com/photos/OQMZwNd3ThU)

没有人为技术面试做准备。毕竟，**你何必呢？**

无论你是刚从大学毕业的研究生，还是有 20 年工作经验的人，你都已经花了数年时间磨练技能，而这些技能只需要展示出来。你已经准备好了，没有任何准备可以代替你已经花在发展技能上的时间。

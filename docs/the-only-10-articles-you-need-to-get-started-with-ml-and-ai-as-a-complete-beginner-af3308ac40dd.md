# 1.数据理解和清理

你可能会感到惊讶，但数据科学家 66–79%的时间都花在获取、理解和清理数据上。

在这些活动中，您将:

1.  从数据源查询数据以形成数据集；
2.  理解数据集，以及每个属性的业务意义；
3.  检查丢失的值，并找出如何处理它们；
4.  使用单变量分析查看单列分布，以纠正偏态并修复异常值；
5.  了解要素间的依赖关系，并使用双变量分析处理多重共线性；
6.  处理分类数据，使其可被模型吸收；
7.  理解属性之间的复杂关系，以及这些关系如何影响业务任务。

作为一个起源于 20 世纪 60 年代的编程概念，面向对象编程现在是当今最流行的编程风格之一。

但是为什么呢？是什么让面向对象编程如此流行？这将试图说明为什么它被认为是如此有用，为什么每个程序员都应该至少了解一点。

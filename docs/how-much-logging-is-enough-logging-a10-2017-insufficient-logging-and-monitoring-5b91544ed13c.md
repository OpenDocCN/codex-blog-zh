# 介绍

乍一看，这似乎不是一个真正的漏洞，而是一个最佳实践，但事实并非如此。如果没有及时检测到正在进行的攻击，或者根本没有检测到，我们的其他安全措施可能会在我们不知道的情况下被篡改。在发生攻击的情况下，我们应该及时得到正确详细程度的通知。

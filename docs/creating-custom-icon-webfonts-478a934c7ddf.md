# 创建自定义图标 web 字体

网络字体图标是现代网络开发的一个重要组成部分，传统上印刷工人称之为字体图标。它们有很多优点:

1.  与 SVG 或传统图像相比，使用更新的格式(如 WOFF 和 WOFF2)压缩文件，文件大小更小。
2.  对这两种更有效格式的支持现在已经足够广泛，我们可以抛弃旧的格式，如 OTF、EOT 或 SVG。
3.  字体是可缩放的，像素化除了改变你的字体大小之外什么也不做。

Drupal 是一个用 PHP 编写的灵活的开源内容管理系统(CMS)。和 WordPress 一样，Drupal 站点是动态的，因为它连接到一个数据库。虽然经常需要开发人员来构建和维护 Drupal 站点，但是创作界面允许您创建内容和配置站点，即使您不是开发人员。

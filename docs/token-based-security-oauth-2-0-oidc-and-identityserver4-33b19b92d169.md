# 介绍

大多数情况下，我们都知道登录 web 应用程序的用户名/密码机制。我们也可能听说过、使用过或实现过认证/授权系统，不管有没有框架。

在这种情况下，基于令牌的安全性是保护后端 API 的常见机制之一。

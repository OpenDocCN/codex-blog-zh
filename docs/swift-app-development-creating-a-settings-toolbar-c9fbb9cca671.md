几乎每个应用程序似乎都有一个“设置”菜单或“收藏夹”标签，菜单出现在现有屏幕的顶部。这些菜单是添加功能的好方法，不需要导航到完全不同的视图。

幸运的是，这些视图并不太难制作，在这篇文章中，我们将详细介绍如何创建“工具栏”和“设置”

![](img/4894f9404bdd385a12da43ea0dcaa936.png)

在 Excel 中存储数据的标准方式是一个单元格对应一个值。然而，一个值可能由多个部分组成，这使得它更难处理。您也可以从网页拷贝内容，但在粘贴时会发现所有内容都被打包到同一个单元格中。

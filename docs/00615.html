和钱有关系吗？

$0、$1等表示对集合中的项目执行动作时的项目。

例如，要打印一个数字列表，可以使用内置的forEach()函数和$0，如下所示:

let numbers = [1, 2, 3, 4, 5]
numbers.forEach { print($0) }

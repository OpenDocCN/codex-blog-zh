# 六角形建筑

六边形架构和干净架构是[以领域为中心的](/codex/clean-architecture-for-dummies-df6561d42c94) [架构模式](https://en.wikipedia.org/wiki/Architectural_pattern)。它们功能强大，能够适应您的需求；甚至像命令行工具这样的简单应用程序也从集中域中受益(将 I/O 与实际算法分开)。

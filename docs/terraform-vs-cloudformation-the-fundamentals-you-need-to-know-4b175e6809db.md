当以协作方式工作时，手动创建和维护云基础架构可能会很有挑战性。在某些情况下，可以将基础设施用作代码(IaC)。

借助 IaC，您的 it 基础架构将更加灵活，使您能够自动执行配置、升级和交付 IT 时执行的操作。

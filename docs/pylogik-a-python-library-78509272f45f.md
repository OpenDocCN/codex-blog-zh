这个 python 库致力于一些图像和统计数据的比较，这是我为自己的工作开发的其他库中没有的，我想我应该与更广泛的社区分享。我希望它是一个包罗万象的名字，并在“逻辑”这个词上发挥作用，因为它包含了一个折衷但有用的函数和类的组合。随着我为…创建各种各样的函数和类，我希望继续增加这个库

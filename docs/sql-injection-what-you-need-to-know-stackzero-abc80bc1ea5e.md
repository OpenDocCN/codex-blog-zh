# 什么是 SQL 注入？

SQL 注入(SQLi)是对利用目标软件中的安全漏洞的 web 应用程序(最著名的还有 [XSS](https://www.stackzero.net/xss/) )的攻击，特别是，它允许攻击者在易受攻击的数据库上执行一些操作。

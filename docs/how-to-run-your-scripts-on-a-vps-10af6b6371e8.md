# 介绍

有时我们想从笔记本电脑上打猎，有时从台式机上打猎，有时甚至从手机上打猎。有时我们想要运行一个小的快速命令，有时我们想要运行所有端口+ TCP 端口的全网络扫描。有时我们想建立一个反向的 shell 连接，有时我们想生成一个漏洞。有时我们在家打猎，有时我们在朋友家打猎。不是每个人都有钱买一台合适的笔记本电脑，其他人有，但更喜欢云的舒适。

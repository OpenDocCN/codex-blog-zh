## [抄本](http://medium.com/codex)

Apache Hadoop 是一个开源软件框架，用于在商用硬件集群上存储和大规模处理数据集。Hadoop 是一个 Apache 顶级项目，由全球贡献者和用户社区构建和使用。它是根据 Apache License 2.0 许可的。

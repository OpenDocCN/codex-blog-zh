GCP 将其 Spark 和 Hadoop 打包在一起，并将其命名为 Cloud DataProc。

过去需要几小时或几天的操作现在只需几秒或几分钟。快速创建云 Dataproc 集群，并随时调整它们的大小，因此您不必担心数据管道超出集群。

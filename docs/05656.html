枚举值是一种巧妙的验证技术，对于API来说可能是有风险的

枚举值是一种利用类型安全的好方法。它们使得原本难以记忆的数字或字符串变得易于使用。不幸的是，它们也会给API开发者带来风险。

当我的团队设计 AvaTax REST API 时，我是enums的忠实粉丝；但是我想和你们分享我们所面临的问题，以及为什么我们的团队在设计锁步平台API 的时候减少了枚举的使用。我们来分析一下。

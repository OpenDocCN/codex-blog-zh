# 云服务模型的类型

> 原文：<https://medium.com/codex/types-of-cloud-service-models-807def8504ed?source=collection_archive---------9----------------------->

云服务模型有几种类型。最受欢迎的是 IaaS、PaaS、SaaS 和功能即服务(FaaS)。这些模型允许您使用一个软件应用程序的单个实例来服务多个客户，每个客户称为一个租户。也可以使用私有云服务。

# 开发者:如何获得晋升

> 原文：<https://medium.com/codex/developers-how-to-get-promoted-4bf8fdf5aa68?source=collection_archive---------6----------------------->

许多软件开发经理很难识别开发人员的天赋。让开发人员变得非常有价值的技能通常不是管理层所看到的技能。经理和开发人员面临的问题是，提拔合适的人和获得提拔可能会受到多种因素的困扰，这些因素与谁更适合这份工作无关。

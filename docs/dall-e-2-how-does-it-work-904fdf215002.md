# 什么是 DALL-E 2？

DALL-E 2 是一个人工智能模型，可以从自然语言的文本描述中生成逼真的图像。

![](img/954a275fd9942278bef81c6df4eaa44f.png)

由文字描述生成的图像:“一碗汤是通向作为数字艺术的另一个维度的入口”(来源:[https://openai.com/dall-e-2/](https://openai.com/dall-e-2/))

除此之外，DALL-E 2 还可以:

*   使用…对现有图像进行逼真的编辑

AppleScript 是那些人们现在不太谈论的脚本语言之一，但它可用于编写多种用途的高级脚本。

它早在 1993 年就被引入了(作为 System 7 的一部分),并且从那以后一直是 Mac OS 的一部分。就我个人而言，我使用苹果脚本来管理文件和自动化测试工作。

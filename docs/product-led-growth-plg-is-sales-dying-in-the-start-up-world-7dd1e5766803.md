# 什么是 PLG(产品导向型增长)？

顾名思义，产品导向型增长是一种走向市场的战略，它依靠产品本身作为吸引、获得和留住用户的手段

以产品为导向的增长有助于公司保持相关性并建立信任，这是数字化成熟的一个基本组成部分…

2021 年，63%的人使用 Chrome 浏览器。所以你会认为这是最好的选择，对吗？不，不是的，原因如下。

我要提到的第一件事不是每个人都关心的，尽管他们应该关心，Chrome 是非常侵犯隐私的。乍一看，你的谷歌浏览器连接到你的谷歌账户似乎很方便，谷歌账户连接到你的谷歌手机，如果你有 Chromebook，它也可以连接到你的笔记本电脑。这很酷，但对于所有这些免费服务，你不是顾客，你是产品。

谷歌是互联网上最大的广告公司之一，他们尽一切努力让你的广告更加个性化，并确保他们从你那里获得最高的点击率。2021 年初，谷歌提出了一个名为 Floc 的广告系统，以取代他们之前的系统，即给每个用户一个唯一的 id，将广告目标直接放入浏览器。其背后的想法是，用户根据他们的兴趣被分成特定的类别。这不仅是谷歌试图通过强迫所有人使用谷歌的 Floc 来垄断广告空间，也是试图使广告跟踪更加准确，因为跟踪器不仅内置于一些网站，还内置于浏览器。幸运的是，其他浏览器拒绝了这个想法，唯一有 Floc 的浏览器是谷歌 Chrome。

我个人推荐 brave，它不仅开放代码给任何人看，还内置了广告拦截器，并且基于 chrome，chrome 也基于 chrome。有些人可能会说，我可以安装自己的广告拦截器，并拥有同样的功能。目前确实如此，但在 2022 年底，谷歌将移除所有使用 V2 清单的 chrome 扩展。

清单 V2 是用于创建 Chrome 扩展的 API 的第二个版本。不幸的是，Manifest V2 已经走到了生命的尽头，将于 2022 年底被谷歌淘汰。从 2022 年初开始，所有提交到 Chrome 网络商店的扩展都必须使用清单 V3。清单 V3 的问题是，它阻止扩展在网站加载前阻止脚本和请求，因此使广告拦截器接近无效。另一方面，Brave 在浏览器中内置了广告拦截器，因此不必通过 Chrome 网络商店。

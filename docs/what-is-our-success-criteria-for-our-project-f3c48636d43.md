# 短篇小说

在我之前的一家公司，我们进行了一次技术团队改革。新领导决定重写我们当前的项目。

在开完所有像 butterfly 或 phoenix 这样很酷的新项目名称的发布会后，我问我们的技术领导，我们项目的成功标准是什么。

答案是，“当我们交付产品管理要求的所有功能时”

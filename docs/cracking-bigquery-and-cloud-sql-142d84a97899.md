## [法典](http://medium.com/codex)

**SQL** (结构化查询语言)是一种数据操作的标准语言，允许您提出问题并从结构化数据集获得洞察力。它通常用于数据库管理，允许您执行将事务记录写入关系数据库和 Pb 级数据分析等任务。

下面，您将学习如何:

*   将数据库与表和项目区分开来。
*   使用`SELECT`、`FROM`和`WHERE`关键字构建简单的查询。

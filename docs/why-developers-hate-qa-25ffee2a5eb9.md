# 为什么开发者讨厌 QA？

> 原文：<https://medium.com/codex/why-developers-hate-qa-25ffee2a5eb9?source=collection_archive---------7----------------------->

QA 是必要之恶，开发者普遍不喜欢。尽管 QA 任务很重要，但它并没有利用开发人员最有价值的技能。此外，让开发人员做 QA 会造成组织效率低下，因为其他更廉价的员工可以做同样的工作。不仅如此，还降低了开发人员的工作满意度。

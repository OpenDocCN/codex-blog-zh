随着开发团队寻找新的方法来保持高效的工作和有效的沟通，**异步沟通**开始流行。异步沟通不需要计划，减少了[上下文切换](https://www.stepsize.com/blog/mofl-how-to-reduce-context-switching-as-a-developer)，并允许记录、回放和重读信息——这对长期项目非常有利。

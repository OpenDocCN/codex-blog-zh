如果你想让你的孩子有一个成功的未来，你必须教他们编程。你可能想知道为什么或什么会激励你的儿子在年轻时学习编程，这被认为是困难的或有时是无聊的，但它将为他们提供比你期望的更多的东西，因为它不仅仅是一门科学，它代表了一种完整的生活方式，而不仅仅是一份工作或一种爱好。

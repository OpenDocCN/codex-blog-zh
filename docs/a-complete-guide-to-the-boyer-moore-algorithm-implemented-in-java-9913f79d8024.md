## 搜索字符串的最快方法

在我的日常生活中，我使用 Control-F 来搜索代码、文章和 API 文档中的关键字，这几乎是我的第二天性。这种体验几乎是即时的，而且总是准确的。人们很容易忽略使这成为可能的实际代码。Boyer-Moore 算法是在给定文本中查找字符模式的最聪明、最快的方法之一。

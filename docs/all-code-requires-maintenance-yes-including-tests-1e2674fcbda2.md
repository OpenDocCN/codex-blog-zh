所有的代码都需要维护，这包括所有的测试代码。

人们很容易认为测试是一次性支出。这些年来，我听到许多工程师说，“我们几乎完成了 UI 测试的编写。”潜台词是，“然后我们将让它们运行并转移到另一个项目。”

OpenStego 是一个隐写术应用程序，它提供两个功能:第一，*“数据隐藏”，*，它可以隐藏封面文档中的任何信息，第二，*“水印”*，它可以用不可检测的标记对文档进行水印处理。它往往被用来区分未经批准的文件复制。

要下载该工具，请访问【https://www.openstego.com/】[](https://www.openstego.com/)**网页。**

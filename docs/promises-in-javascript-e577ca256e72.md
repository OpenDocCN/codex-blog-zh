构造函数将一个叫做执行函数的函数作为参数。

一个承诺有以下内部属性:
**状态:这个属性有三个值。待定、已完成和已拒绝。**

1.  **待定-
    -**当执行程序功能开始执行时。
2.  履行-当承诺被解决。
3.  拒绝-当承诺被拒绝。

**结果——这个属性又有三个值。**

1.  未定义—当状态为待定时。
2.  价值——承诺兑现时(价值)称为价值。
3.  错误——承诺被拒绝时(错误)称为错误。

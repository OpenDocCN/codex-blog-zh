# 构建和使用虚拟机

虚拟机也称为 VM，是运行在现有操作系统或主机操作系统之上的操作系统。虚拟化操作系统通常被称为来宾，由一个名为虚拟机管理程序的软件管理。虚拟机管理程序允许您基于您的客户机管理和分配 CPU、内存和磁盘等资源。您可能听说过的流行虚拟机管理程序包括微软的 Hyper-V、甲骨文的 VirtualBox、VMware 和 KVM。通常，当您听到“云”这个术语时，它通常只是指在某个数据中心的其他服务器场的专用虚拟机管理程序上运行的数千个虚拟机。

虚拟机让您变得与平台无关，这意味着您不再局限于任何特定的操作系统和可用的工具。很多人把所有的时间都花在一个 OS 上，争论哪个更好。我来告诉你，没关系。作为一名专业人员，你应该精通所有这些技术，并且能够自如地使用任何平台。这最大限度地扩大了你的技能范围，使你在任何情况下都有效。虚拟机还可以让您在隔离的环境中灵活地进行培训和研究，而不会影响您的主驱动程序。您可以快速拍摄来宾操作系统的快照，并将其恢复到之前的状态，打开您收到的可疑文件。使用虚拟机。想练习配置服务器吗？使用虚拟机。攻击目标？使用虚拟机。

# 学习命令行

不要逃避，要拥抱。命令行界面通常被称为 shell，它是与操作系统交互的最简单也是最有效的方式。例如，如果我在大小超过 1GB 的 CSV 文件中查找条目，Excel 可能会冻结或崩溃。运行一个轻量级的 shell 命令可以让我立刻得到我需要的东西。为什么叫贝壳？操作系统运行的重要部分被称为“内核”，因为它是系统的中心。暴露给用户的部分称为外壳，因为它包裹着内核。这就像一辆汽车，方向盘踏板，仪表板，让你控制发动机或传输都存在于汽车的外壳。

命令行为您提供了对操作系统附带的软件功能的最低级别的访问。许多最有用的工具都没有可以点击的图形用户界面(GUI)。掌握命令行可以扩大您的武器库，让您事半功倍。它允许您能够使用脚本和自动化来处理重复的任务，否则会浪费大量的时间。通过学习命令行来学习自动化您的工作流会使您成为团队中非常有价值的资产。我建议从 Bash 或重生的 Shell 开始，因为几乎每个 Linux 发行版都默认提供了它。MacOS 在其终端应用程序中使用了它，该应用程序后来改为 Zsh 或 Z-Shell，并具有一些更好的功能。如果说 Bash 像一辆丰田，那么 Zsh 更像一辆雷克萨斯。Bash 是如此受欢迎和有效，以至于微软发布了用于 Linux (WSL)的 [**Windows 子系统，并让你安装几个不同的 Linux 版本来使用 Bash 作为原生应用。这非常方便，因为您可以访问所有的 Linux 工具，而不必切换到虚拟机。现在我们明白了，Powershell 实际上是 Windows 的首选本机 shell。这与您使用 Linux 命令行的方式非常不同，但是它为您提供了大量强大的 Windows 管理功能。如果您在以 Windows 为主要操作系统的环境中工作，那么您肯定也应该学习 PowerShell。**](https://docs.microsoft.com/en-us/windows/wsl/)

# 掌握系统管理的艺术

系统管理涉及计算机的配置和维护，无论是个人设备还是高性能服务器。系统管理就是从里到外了解您的平台和各种工具，以便能够帮助那些不了解的人。无论你的技能是什么，试着在实践中摸索和学习。删除一些文件，并尝试恢复它们。用类似[**Windows Sysinternals**](https://www.techtarget.com/searchwindowsserver/definition/Windows-Sysinternals)这样的工具下载/打开并监控虚拟机中的旧病毒，看看它们做了什么。尝试在不知道登录信息的情况下从计算机中提取文件和密码。不管它是什么，通过阅读指南并跟随它，推动你已经知道的极限。每天多练习一点，你很快就会提高。

# 计算机连网

这是它的核心和灵魂。它是对设备如何相互作用以及数据如何从 A 点到 b 点的理解。强大的基础和网络将使您成为 rockstar 故障排除员。无论你是红队队员，防守队员，还是日常跑步队员，它都能发挥作用。

两个概念模型支配着计算机网络:TCP IP 和 OSI。他们将所有不同的网络和电信协议分成不同的层。TCP/IP 比较老，使用四层:网络接入层、互联网传输层和应用层。OSI 代表开放系统互连，由国际标准化组织(ISO)开发。这些家伙定义了从国家代码到时间和日期格式的一切。OSI 较新，使用七层:物理数据链路层、网络传输层、会话层和应用层。所有这些层只是描述正在发生的事情和地点的一种方式。

因此，如果你收到一个来自不同国家的人的包裹，它将在信封、盒子、车辆和飞机之间传递，每个都有其寻址方法和操作程序。当邮局告诉你有航空公司的问题时，你就知道是运输系统的哪个环节延误了你的投递。同样，网络层都有不同的功能，但它们是一个整体，让你可以将视频从机架中的服务器传输到你手中的设备。了解每一层发生了什么，让你看到矩阵，并在你的工艺上更加熟练。

# 个人数字安全

这是我特别热衷的一个领域，因为它影响着我们的家庭、朋友和组织。网络犯罪产业正在蓬勃发展。你不必滚动很远就能看到网上黑市是什么样子。随着技术越来越多地与我们的生活交织在一起，从联网汽车到冰箱，漏洞和攻击载体将越来越多。如果你想深入研究网络安全，没有比从自己开始更好的地方了。从密码和加密到安全通信，了解最新的安全新闻和最佳实践。你可能只是你办公室里的主题专家，其他人会向你寻求建议。这个建议可能会保护你的公司不成为头条新闻。但最重要的是，你也要活出自我！大多数时候，网络入侵源于安全运营或 IT 部门，仅仅是因为人们不了解或不实践基本的数字卫生。

# 什么是代码重构？

[代码重构](https://www.stepsize.com/blog/the-ultimate-engineers-guide-to-refactoring)是在不改变代码功能的情况下，对之前编写的代码进行重构的过程。重构并不意味着添加新功能或重新编写代码来修复任何类型的错误。

进行重构有几个好处，包括:

*   表演
*   更好的代码覆盖率
*   提高代码可读性
*   更好地理解…

# 走向自文档化代码

> 原文：<https://medium.com/codex/towards-self-documenting-code-371364bdccbb?source=collection_archive---------2----------------------->

## 在大多数情况下，如果你依靠注释来解释代码，你就做错了。我将向您展示一些实现[自文档化代码](https://en.wikipedia.org/wiki/Self-documenting_code)的方法。不，这不是乌托邦。

这不是反对代码注释的情况。这是一个支持不言自明的代码库的例子。作为副作用，注释变得多余。使用代码交流被 [XP](https://en.wikipedia.org/wiki/Extreme_programming) 社区广泛提倡。

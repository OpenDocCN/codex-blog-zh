# 介绍

Databricks notebook 是一款基于网络的 Jupyter 笔记本，由 Spark clusters 提供支持。数据工程师、科学家和分析师在 Databricks 生态系统中使用 Databricks notebook 执行 ETL、可视化和机器学习模型培训等任务。在本文中，我们研究了 Databricks notebook 中一些有用的生产力特性，以促进您的数据科学和工程工作流。

# CAGradientLayer

CAGradientLayer 是一种 CALayer 类型，您可以在其中创建渐变。您可以使用任意多的颜色，并修改其他属性，如角度和位置。您可以使用它来创建应用程序背景的渐变或自定义按钮的外观。

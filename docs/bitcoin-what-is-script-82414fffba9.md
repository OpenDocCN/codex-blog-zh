# 比特币:什么是脚本？

> 原文：<https://medium.com/codex/bitcoin-what-is-script-82414fffba9?source=collection_archive---------19----------------------->

脚本是比特币的智能合约编程语言。

你可以认为它类似于以太坊的坚固性。

就本文而言，大写字母“S”的“script”是编程语言的名称，“Script”是一组编码指令。

# 介绍

`CFArray`是一个核心基础类型(ref: [这里是](https://developer.apple.com/documentation/corefoundation/cfarray?language=objc))，可以用来存放任何类型的对象集合。它是一个静态的、不可变的数组，这意味着您不能在数组中添加或移除元素。

大多数网站需要某种形式的用户输入。我们每天遇到的例子包括搜索栏、登录请求和填写联系信息。

所有这些表单都以 HTML 开始，然后用 CSS 样式化，最后用后端编程语言处理输入的表单数据——例如 PHP 或 Javascript。
